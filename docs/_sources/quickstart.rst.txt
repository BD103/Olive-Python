Quickstart
==========

Prerequisites
-------------

Things that you will need to start developing a Python package:

1. `A copy of Python 3 installed on your computer <https://www.python.org/downloads/>`_ (This guide is written in 3.8)
2. General knowledge coding in Python (I recommend `W3Schools <https://www.w3schools.com/python/default.asp>`_)
3. A peek at the `Python docs <https://docs.python.org/3/>`_ and a good read of the `global module index <https://docs.python.org/3/py-modindex.html>`_
4. A code development platform

  - This could be a folder on your computer
  - `VSCode <https://code.visualstudio.com/>`_
  - `Replit.com <https://replit.com>`_ (Recommended)
  - `PyCharm <https://www.jetbrains.com/pycharm/>`_
  - Some other IDE

5. A glass of water and some snacks (Optional)
6. A book for when I bore you to death with lists (Optional)

8. A copy of `Git <https://git-scm.com/downloads>`_ (Required for Windows or if you want to use source control)
