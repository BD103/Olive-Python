Choose Your Path#

Now that you’ve learned what a package is and how it works, it’s time for you to make a decision.

There are many tools out there that help you with making a Python package. You have to decide whether you want to use these tools or stick to the classic approach.

Tools#

There are many tools out there, but we currently only have a tutorial for one.

  • Poetry (Recommended)

Classic#

Setuptools is the classic method for developing Python packages. Though not the recommended method, it is good to know how it works. Many pre-existing packages uses Setuptools, so knowing how it works helps you read the source code.

  • Setuptools
